# Shrikant Cyber Portfolio

A futuristic single-page cybersecurity portfolio designed for GitHub Pages.

## How to publish on GitHub Pages

1. Create a new public repository named `portfolio` or `<your-github-username>.github.io`.
2. Upload all files from this folder.
3. Go to Repository Settings > Pages.
4. Under Build and deployment, select `Deploy from a branch`.
5. Choose `main` branch and `/root` folder, then save.
6. Update the LinkedIn button in `index.html` by replacing `REPLACE_WITH_PUBLIC_LINKEDIN_URL` with your public LinkedIn profile URL.

## Customize

- Replace `assets/avatar.png` with a new avatar if needed.
- Edit service text in `index.html`.
- Update email, GitHub, LinkedIn, projects, and resume links.
- Change colors in `style.css` under the `:root` variables.
